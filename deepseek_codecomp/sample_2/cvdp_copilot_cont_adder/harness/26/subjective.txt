When ACCUM_MODE is set to 1, the module accumulates data points until the number of samples reaches the value specified by window_size. Each valid data input increments the sample_count register. Once sample_count equals window_size, the sum_accum register holds the accumulated sum of the last window_size samples, and the sum_ready_reg signal indicates that the sum is ready. At this point, the sum_out and avg_out signals are updated with the current sum and average, respectively. After the sum is computed, both sum_accum and sample_count are reset to zero, preparing the module to accumulate the next set of samples.
