### Module Description:

The `unpack_one_hot` module processes packed data stored in a 256-bit source register (`source_reg`) and expands it into a 512-bit destination register (`destination_reg`). The unpacking is controlled by a 3-bit one-hot selector (`one_hot_selector`), a sign control signal (`sign`), and a size control signal (`size`). The module supports unpacking at different granularities, handling both signed and unsigned data.

#### Inputs:

3. **`one_hot_selector [2:0]` (3 bits)**:

       - If `size` is `0`, treat each 4-bit segment of `source_reg` as an unpacked 8-bit value.
4. **`source_reg [255:0]` (256 bits)**:
   - The packed source register containing the input data to be unpacked.

#### Outputs:
1. **`destination_reg [511:0]` (512 bits)**:
