Provide me one answer for this request: Design an RTL module that implements the Hill cipher encryption algorithm with the following specifications:

- **Block Size**: 3 letters (each letter represented by 5 bits).

- **Modular Arithmetic**: All operations are based on modulo 26.

The Hill cipher is a classical symmetric encryption algorithm based on linear algebra. It operates on blocks of plaintext and uses matrix multiplication to produce ciphertext.

### 1.1. Basic Concepts

- **Plaintext**: The original message to be encrypted.
- **Ciphertext**: The encrypted message.
- **Block Size**: The number of letters processed at a time. The design has to use a block size of **3**.
- **Key Matrix**: A square matrix used for encryption. For a block size of 3, the key matrix is a 3x3 matrix.
- **Modular Arithmetic**: All calculations are performed on modulo 26, corresponding to the 26 letters of the English alphabet.

### 1.2. Encryption Process

1. **Convert Letters to Numbers**: Map each letter to a number (e.g., A=0, B=1, ..., Z=25).
2. **Form Plaintext Vector**: Group the plaintext into blocks of 3 letters and represent them as column vectors.
3. **Matrix Multiplication**: Multiply the key matrix by the plaintext vector.
4. **Modulo Operation**: Apply modulo 26 to each element of the resulting vector.
5. **Convert Numbers to Letters**: Map the resulting numbers back to letters to obtain the ciphertext.
