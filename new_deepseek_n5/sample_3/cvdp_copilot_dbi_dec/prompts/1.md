A **Data Bus Inversion (DBI) decoder** is a digital circuit used to reduce power consumption and electromagnetic interference (EMI) in high-speed digital buses. DBI decoder decodes a group of data bits using an additional bit called a control bit, which indicates whether the current data bits received over the bus are in inverted form or not.

For example, in the case of a DBI decoder that processes 40 data bits, the 40-bit input data is divided into two 20-bit groups. Each 20-bit group is associated with a control input signal, and if this control signal is high, the data in the corresponding 20-bit group is inverted; otherwise, no operation is performed on it.
