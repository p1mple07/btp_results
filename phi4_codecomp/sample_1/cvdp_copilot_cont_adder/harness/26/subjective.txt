In ACCUM_MODE=1, the module accumulates the weighted input by adding it to sum_accum and increments sample_count on each valid data input. The window_size parameter defines the number of samples required before the accumulation window is considered complete. When sample_count + 1 reaches or exceeds window_size, the combinational logic sets sum_ready_reg to 1, indicating that the accumulation window is filled. At that point, the module updates the output registers with the accumulated sum and its average, and then resets sum_accum and sample_count to zero to start a new accumulation cycle.
