The section in the test plan that describes the comparison between the Golden Model and the DUT outputs is the 'Output Verification' section under 'Test Methodology'. In this section, the test plan explains that after applying inputs and computing the expected outputs (model_I and model_Q) using the Golden Model, the testbench compares these expected values with the DUT outputs (I and Q). It details how each output component is checked, how mismatches are logged (including the index, DUT value, and expected value), and how the test counters (total, passed, and failed tests) are updated based on the results. This directly corresponds to the implementation in the check_outputs task of the tb_qam16_mapper_interpolated testbench.
