In ACCUM_MODE=1, the continuous_adder module continuously accumulates the weighted input into the sum_accum register while incrementing the sample_count register with each valid data sample. The module uses the window_size parameter to determine when the number of accumulated samples reaches a specified threshold. Once the condition (sample_count + 1 >= window_size) is met, the module sets the sum_ready_reg flag, signaling that the accumulation is complete. At this point, the sum_accum and sample_count registers are reset to zero to begin a new accumulation cycle, ensuring that each window of samples is processed independently.
