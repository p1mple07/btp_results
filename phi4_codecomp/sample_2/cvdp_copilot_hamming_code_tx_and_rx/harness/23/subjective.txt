The part of the testbench that generates a single-bit error is in the send_data task. It first copies the original Hamming code into modified_data_out, then selects a random bit position (between 1 and ENCODED_DATA-1) using $urandom_range, and finally toggles that bit with modified_data_out[bit_pos] = ~modified_data_out[bit_pos]. This simulates a realistic transmission error by introducing a single-bit error.
