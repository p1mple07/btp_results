The golden model is implemented in the function calculate_expected_outputs, which computes the expected outputs as:

  expected_re = (i_cos * i_data_re) - (i_sin * i_data_im)
  expected_im = (i_sin * i_data_re) + (i_cos * i_data_im)

This function is then used in the task apply_inputs_and_check to compare the DUT outputs with these reference (golden) values.
