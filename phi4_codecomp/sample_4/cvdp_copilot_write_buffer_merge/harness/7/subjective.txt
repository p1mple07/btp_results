The testbench determines the end of simulation by checking two conditions: it stops when the number of input writes (num_inputs) equals the expected count (num_inputs_expected, which is num_outputs_expected multiplied by BUFFER_DEPTH) and when the number of output write enables (num_wr_en_out) reaches the expected number (num_outputs_expected). This dual condition ensures that the simulation ends either when all input writes have been processed or when the expected number of output writes has been observed.
