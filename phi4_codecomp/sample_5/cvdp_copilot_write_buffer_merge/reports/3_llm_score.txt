The merged_data signal acts as an accumulator that combines multiple write operations into a single wide data output, effectively merging several smaller data words into one. With each incoming write, the module shifts the existing merged_data right by INPUT_DATA_WIDTH bits and concatenates the new write data on the left, thereby updating merged_data to represent the accumulated writes.
